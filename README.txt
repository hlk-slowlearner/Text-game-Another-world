[游戏名]：Another World（荒野篇）
[游戏类型]：小型文字冒险
[游戏简介]：作为主角的你失去了记忆，出现在了异界一片怪物肆虐的荒野上。请在每个命运的分岔口做出谨慎的选择，以生存或是征服为目的，走出这片危险的荒原。
[游戏初衷]：圆我儿时的勇者梦。
[运行环境]：macOS系统
[开发环境]：python3.8
注：【】内文字是一些初步的改进建议，但代码还未完成。

游戏框架:

	游戏开始 → 回答问题，根据不同的回答产生不同属性模板。【这里的问题机制可以更复杂些】
	   |
	   |
	  序章 → 异界第一战，不同的选择得到不同的新手礼包，并让玩家了解对战方式。【新手奖励可	   |	以更多样化】
	   |
	  荒原 → 玩家通过移动人物来试图走出荒野，每次移动一步都有可能遇到不同的野怪、事件，
	   |    同时在地图的八个固定位置有八个唯一的特殊事件，玩家抵达就会触发（仅一次）。
	   |	【可以在地图上创建一些无法通过的建筑物，或是密道】
	  尾声 → 荒原篇结束
-----------------------------------------------------------------------------
游戏人物属性：
	生命上限
	血量
	攻击:决定人物每次进攻造成的伤害，若人物攻击小于敌方护甲则直接失败
	护甲:决定人物受到伤害后抵消的值
	速度:决定先手权以及逃跑的成功率
	悟性：悟性越高，越容易触发双倍战斗奖励
	装备：类似成就
	称号：类似成就
	状态：增加趣味性
	吸血：传奇属性，特定事件后触发。
-----------------------------------------------------------------------------
战斗环节：
	回合制战斗，战斗胜利后可以选择战斗奖励（属性点加成）【战斗奖励的门槛可以更高】
	失败后可以读取存档，回到上次遇见”生命之泉“的地方。
